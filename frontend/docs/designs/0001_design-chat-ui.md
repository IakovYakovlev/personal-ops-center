**Дата:** 24.03.2026

## Контекст

Дизайн для сервиса Neural Assistant: окно чата с ИИ, поддержка стриминга ответов, интеграция в Dashboard.

## Ссылка на макет

[Figma](https://figma.com/...)

## Требования

- Стриминг ответов
- Мобильная адаптация
- Интеграция с существующим UI
- Поддержка темной темы

## Варианты

- Вариант 1: Классический чат с аватарами, фиксированное поле ввода
- Вариант 2: Минималистичный чат, плавающее окно

## Итоговое решение

Выбран классический чат с аватарами, адаптивной версткой и поддержкой стриминга. Основной акцент на UX и скорости отклика.

## Примечания

- Для стриминга использовать Server-Sent Events или WebSocket
- UI реализовать на ShadCN UI
- Учесть плавную прокрутку и автофокус поля ввода
